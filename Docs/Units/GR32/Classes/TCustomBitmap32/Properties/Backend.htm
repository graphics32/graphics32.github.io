TCustomBitmap32.Backend

property Backend: TBackend;

Description

Returns or sets the current back-end used for the bitmap's surface and additional graphics subsystem specific features that are accessible via interfaces.

Note: Interface queries on the current bitmap will also be delegated to the back-end.
