IndexGR32SetAlpha


SetAlpha

function SetAlpha(Color32: TColor32; NewAlpha: Integer): TColor32;

Description

Returns a color with the new alpha channel. This function automatically performs clipping of the NewAlpha parameter to [0…255] range.
