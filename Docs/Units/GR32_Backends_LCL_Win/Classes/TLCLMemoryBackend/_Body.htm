TLCLMemoryBackend

Ancestors

TMemoryBackend,IPaintSupport,IDeviceContextSupport

Description

This backend keeps the backing buffer entirely in memory and offers IPaintSupport without allocating a GDI handle

Features

  • Access to the surface via GDI routines (via IDeviceContextSupport);
  • Supports painting via IPaintSupport

See Also

IDeviceContextSupport, IPaintSupport
