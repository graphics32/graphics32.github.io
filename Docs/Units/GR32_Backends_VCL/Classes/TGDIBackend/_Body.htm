TBackend

Description

This back-end allocates the memory and resources required for the surface directly via the Windows GDI subsystem.

Features

  • Access to the surface via GDI routines (via IDeviceContextSupport and IBitmapContextSupport);
  • Access to the surface via the default VCL canvas (via ICanvasSupport);
  • Support for text drawing and measuring and related font methods (via ITextSupport and IFontSupport);
  • Supports all of the other interfaces required for TBitmap32, making it the default back-end for this class.
