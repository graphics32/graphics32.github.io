TClipperBase.Clear

procedure Clear;


This method allows the TClipper object to be reused when clipping operations are required on different polygon sets. (If different clipping operations are performed on the same polygon set, then Clipper's Execute method can be safely repeated without clearing and re-adding polygons.)
