TBarycentricGradientPolygonFiller.SetColorPoints

procedure SetColorPoints(ColorPoints: TArrayOfColor32FloatPoint); override;

procedure SetColorPoints(Points: TArrayOfFloatPoint; Colors: TArrayOfColor32); override;

Description

SetColorPoints can be called to set all color points at once. Either specify the points and colors seperately or combined as TArrayOfColor32FloatPoint. No more than 3 points may be passed, otherwise an exception will be raised.
