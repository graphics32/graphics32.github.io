TColor32Gradient.GetColorAt

function GetColorAt(Offset: TFloat): TColor32;


Description

GetColorAt returns the color at a specified offset. The color is clamped to the start color or end color if the offset is outside the bounds. Between two specified colors/offsets that are stored in the TColor32Gradient object, the color is interpolated.
