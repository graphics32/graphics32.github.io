TCustomGradientLookupTablePolygonFiller.Create

constructor Create(LookupTable: TColor32LookupTable); overload; virtual;

Description

Creates and initializes an instance of TCustomGradientLookupTablePolygonFiller.

After calling the inherited constructor, Create initializes:

  • GradientLUT to the passed TColor32LookupTable object
  • UseLookUpTable to true
