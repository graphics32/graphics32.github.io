TCustomRadialGradientPolygonFiller.EllipseBounds

property EllipseBounds: TFloatRect; //read and write

Description

The EllipseBounds specifies the outer bounds of an ellipse used by radial polygon filler. The center and radius will be calculated internally based on the given ellipse bounds
