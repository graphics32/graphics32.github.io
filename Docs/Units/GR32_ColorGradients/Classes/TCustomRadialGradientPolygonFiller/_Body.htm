TCustomGradientPolygonFiller

Description

TCustomRadialGradientPolygonFiller is a common ancestor for objects that can provide color gradient samples. The color gradient itself is defined in the Gradient property. The WrapMode property defines how the gradient continues beyond its specified bounds
