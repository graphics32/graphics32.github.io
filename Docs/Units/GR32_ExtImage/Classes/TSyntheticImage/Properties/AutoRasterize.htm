TSyntheticImage.AutoRasterize

property AutoRasterize: Boolean;

Description

If AutoRasterize then the synthetic image will rasterize when the associated rasterizer has changed, otherwise Rasterize has to be called in order to start rasterization. AutoRasterize is set true upon creation.
