TCustomImage32.ScaleX

property ScaleX: Single;

Description

Specifies the X scale of the bitmap image when ScaleMode is smScale or smOptimalScaled.

Warning: do not set too small scales in baTile BitmapAlign mode, as this can have severe impact on performance.
