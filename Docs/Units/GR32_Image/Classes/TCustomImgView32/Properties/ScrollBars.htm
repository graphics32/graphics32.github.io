TCustomImgView32.ScrollBars

property ScrollBars: TIVScrollProperties;

Description

Controls appearance of scroll bars.

The scroll bars in TCustomImgView32 are of the TCustomRangeBar type. Use ScrollBars to control properties of the vertical and horizontal scroll bar simultaneously.
