IndexGR32_ImageTIVScrollPropertiesSize


TIVScrollProperties.Size

property Size: Integer;

Description

Simultaneously controls the Height of horizontal and Width of veritical scroll bars.

If Size is 0 (default), the corresponding size is set to the default one, specified in current desktop settings.
