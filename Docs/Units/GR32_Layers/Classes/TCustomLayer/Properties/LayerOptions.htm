TCustomLayer.LayerOptions

property LayerOptions: Cardinal;

Description

A 32-bit unsigned integer value composed of Layer Options Bits, which controls the layer's behavior.

Note that the lower 24-bits of LayerOptions have no predefined meaning; they are available for customization purposes, like layer categories, etc.
