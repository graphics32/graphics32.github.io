TCustomLayer.MouseEvents

property MouseEvents: Boolean;

Description

Returns True if the LOB_MOUSE_EVENTS bit in LayerOptions is set; otherwise False is returned. Setting MouseEvents changes the LOB_MOUSE_EVENTS bit in LayerOptions accordingly.
