TPositionedLayer.Scaled

property Scaled: Boolean;

Description

Specifies if the layer is scaled and determines the result returned by GetAdjustedLocation. When set to True, scaling is influenced by the layout of bitmap image in the container (TCustomImage32 object or its descendant).
