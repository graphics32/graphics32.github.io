TRubberbandLayer.FrameStippleCounter

property FrameStippleCounter: Single;

Description

Specifies the current position of the stipple pattern of the surrounding frame.

Hint: By using a timer and modifying the FrameStippleCounter, it is possible to animate the frame of the rubberband.
