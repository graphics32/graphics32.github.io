TArrowBar.ShowHandleGrip

property ShowHandleGrip: Boolean;

Description

Determines visibility of a handle grip:

Although TArrowBar itself does not have any handles, this property is used in descendants TCustomRangeBar and TCustomGaugeBar.

Note, that this property is available only when Style=rbsMac.
