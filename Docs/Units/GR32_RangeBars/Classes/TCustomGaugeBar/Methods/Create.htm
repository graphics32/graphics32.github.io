TCustomGaugeBar.Create

constructor Create(AOwner: TComponent); override;

Description

Creates and initializes an instance of TCustomGaugeBar.

After calling the inherited constructor, Create initializes the following properties:

  • HandleSize to 0;
  • LargeChange to 1;
  • Max to 100;
  • Min to 0;
  • Position to 0;
