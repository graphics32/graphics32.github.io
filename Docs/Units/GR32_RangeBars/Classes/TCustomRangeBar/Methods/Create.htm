TCustomRangeBar.Create

constructor Create(AOwner: TComponent); override;

Description

Creates and initializes an instance of TCustomRangeBar.

After calling the inherited constructor, Create initializes the following properties:

  • Centered to False;
  • Increment to 8;
  • Position to 0;
  • Range to 0;
