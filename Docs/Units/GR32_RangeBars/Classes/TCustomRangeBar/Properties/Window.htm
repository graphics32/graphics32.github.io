TCustomRangeBar.Window

property Window: Single;

Description

The size of the associated visible area:

In case the value of the property is set to zero, range bar assumes that it is equal to its own size, as shown in the picture above.

Using this property is required when the size of the scrollable window is different from the size of the range bar.
