TCustomRangeBar

Description

TRangeBar is a control similar to standard TControlScrollBar. Normally it is associated with some other control that has a scrollable region.

TRangeBar implements the generic behavior introduced in TCustomRangeBar. It publishes some inherited properties from TCustomRangeBar, but does not introduce any new behavior.
