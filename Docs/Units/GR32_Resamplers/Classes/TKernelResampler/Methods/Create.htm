TKernelResampler.Create

constructor Create(Bitmap: TBitmap32); override;

Description

Call Create to construct a new TKernelResampler instance.

After calling the inherited constructor, the following properties are set:

  • TableSize is set to 32;
  • Kernel is set to TNearestKernel;
