Description

The bloat transformation class provides an adjustable type of lens distortion. Setting bloat power to a positive value gives a fisheye-like distortion, but with less spheric effect (as seen in the left image below). Setting the bloat power to a negative value will affect the transformation in the opposite way (as seen in the right image below).
