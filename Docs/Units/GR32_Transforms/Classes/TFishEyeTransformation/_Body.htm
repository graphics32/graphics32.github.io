Description

The fisheye transformation implements a transformation similar to wideangle lens distortion ("fisheyes"). The effect of the transformation can be seen in the image below. For illustrative purposes the distortion were amplified by rasterizing the transformation to the vector map of TRemapTransformation and adjusting the scale factors of the latter.
