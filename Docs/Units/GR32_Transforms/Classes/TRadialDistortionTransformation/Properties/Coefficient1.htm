TRadialDistortionTransformation.Coefficient1

property Coefficient1: TFloat;

Description

Coefficient for the second order term, usually a small positive or negative number (abs(Coefficient1) << 1 ). A positive value gives barrel distortion, a negative value corresponds to pincushion distortion.
