TRadialDistortionTransformation.MapElements

property MapElements: Integer;

Description

Specifies the number of elements for the discrete map used for the reverse tranform. If it is lower than or equal to 1, a sensible default, depending on the source resolution is used. To increase quality, a higher value may be set.
